Simple Pong Game
  - 2 player game
  - Game will end when a player reaches 3 points

Missing implementation:
  - No way to official Start game - game runs without user input
  - No way to officially end game - game will stay on end screen loop until program is closed

Known bugs:
  - Ball will get stuck inside paddle
  - Ball will pass through top and bottom of paddles
  - Ball will bounce off backside of Paddle 1 
  - End screen text will flicker
